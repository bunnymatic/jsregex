Javascript Regex Playground

by 2rye

how to

  1. Type some javascript regular expression in the regex input box. You don't need to add leading and trailing slashes. That's done on the automatic.
  2. Type a sample string to match against in the other box.
  3. Check out the resulting matches.
  4. Repeat steps 1-3 until you've figured out the right regular expression.
